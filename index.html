SENTINEL — Honeypot Grid

Cowrie SSH honeypot with AI-generated responses (HoneyAI), HTTP canary tokens, and a full aggregation API. Every attacker interaction is logged, TTP-tagged with MITRE ATT&CK, and streamed to the dashboard.

SIEM — Suricata + Wazuh

Live IDS/IPS alert feed from Suricata EVE JSON + Wazuh integration. Rules-based detection, custom SID alerting, passive traffic analysis, and correlation with honeypot events for full kill chain reconstruction.

Threat Detection

NOX-BRAIN — AI Analysis

Local AI (Ollama) grounded on live SENTINEL, IDS, and C2 context. Ask it to analyze threat patterns, generate reports, or automate investigations. Routes tasks across multi-machine AI fleet via ShadowBridge AI queue.

AI-Powered
🌍

OSIRIS — OSINT Engine

Global open-source intelligence dashboard. Domain enumeration, IP reputation, breach data, social footprint — all aggregated into a single interface. Feeds directly into NOX-COMMAND operations.

Intelligence
☠️

Villain C2 — Command & Control

REST bridge over Villain C2 framework. Manage sessions, generate staged payloads, stream loot — all through a clean API. Full implant lifecycle from generation to exfil, visible in the dashboard.

Offensive
🔐

VAULT — Credential Hub

Async hash cracking with John the Ripper and hashcat. RockYou, FastTrack, custom wordlists. Crack queue with real-time job progress, loot storage, and integration into C2 and kill chain phases.

Credential Intel

ATLAS — MITRE ATT&CK

Purple team TTP tracker built on the MITRE ATT&CK framework. All 14 tactics, 200+ techniques. Tag every observed event with red/blue/purple status, auto-populated from SENTINEL and SIEM events.

Purple Team

NOX-COMMAND — Kill Chain

Orchestrate full attack operations: OSINT → Recon → Vuln Scan → Exploitation → Stress Test → C2 → Loot → Report. Each phase plugs into the relevant module — nmap, Nuclei, Villain, VAULT, Impulse DoS toolkit.

Orchestration

HexStrike-AI — CTF Arsenal

104 security tools with intelligent parameterization. nmap, sqlmap, nikto, ffuf, hashcat, hydra, msfconsole, and 97 more — wrapped in an AI layer that picks the right flags and interprets output automatically.

RuView — WiFi Presence

Detect presence and vital signs (heart rate, respiration) from WiFi signals — no camera required. Real-time sensing over WebSocket, REST API for integration. Passive, invisible, uncancellable.
