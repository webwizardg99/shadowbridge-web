Csatlakoztasd a hálózatod a ShadowBridge-hez

Az sb_push.py daemon 30 másodpercenként küldi a géped és szolgáltatásaid állapotát a dashboardra. Kövesd az utasításokat az operációs rendszeredhez.

1
Python telepítése
Ha már van Python, ugorj a 2. lépésre

Töltsd le a Pythont az alábbi linkről, és telepítésnél pipáld be az "Add Python to PATH" opciót!

⬇ Python letöltése
Fontos: A "Add Python to PATH" jelölőnégyzet nélkül nem fog működni a python parancs. Telepítés után zárd be és nyisd újra a PowerShellt.
2
sb_push.py letöltése és csomagok telepítése
PowerShell (nem szükséges Admin)
Windows PowerShell
# Letöltés Invoke-WebRequest https://shadowbridge.store/sb_push.py -OutFile sb_push.py # Csomagok telepítése pip install aiohttp psutil
3
Token beállítása
Nyisd meg Notepad-del, módosítsd a 2 sort
sb_push.py — Notepad-del szerkeszd
# Nyisd meg: notepad sb_push.py # Módosítsd ezeket a sorokat: PUSH_SECRET = "IDE_IRD_A_TOKENED" # Setup / Token panelről NODE_ID = "windows-pc" # bármilyen egyedi név
A tokenedet a ShadowBridge → 🔑 Setup / Token panelen találod bejelentkezés után.
4
Teszt futtatás
Ellenőrzd, hogy működik-e
Windows PowerShell
python sb_push.py --once [15:42:07] Push OK | services:0/0 | sentinel_ssh:0 | vault:0 | ids:0 | ruview:0p
Push OK = sikeresen csatlakozott. A dashboard azonnal frissül.
5
Automatikus indítás (opcionális)
Feladatütemező — percenként push-ol, gép indításakor is
PowerShell — Admin jogosultsággal futtasd
# Feladatütemezőbe felveszi, 1 percenként fut $pythonPath = (Get-Command python).Source $scriptPath = "$PWD\sb_push.py" schtasks /create /tn "ShadowBridge Push" /tr "$pythonPath $scriptPath --once" /sc minute /mo 1 /f SUCCESS: The scheduled task "ShadowBridge Push" has successfully been created.
Eltávolítás: schtasks /delete /tn "ShadowBridge Push" /f
1
Python és csomagok
Kali, Ubuntu, Debian alapú rendszereken
bash
# Python 3 (általában már telepítve van) python3 --version # Csomagok telepítése pip install aiohttp psutil # Ha hibát kap: pip install aiohttp psutil --break-system-packages
2
sb_push.py letöltése és konfigurálása
bash
wget https://shadowbridge.store/sb_push.py -O ~/sb_push.py # Token és node név beállítása nano ~/sb_push.py # Módosítsd: PUSH_SECRET = "IDE_IRD_A_TOKENED" NODE_ID = "linux-node"
3
Teszt és automatikus indítás
bash
# Teszt python3 ~/sb_push.py --once [15:42:07] Push OK | services:0/0 | sentinel_ssh:0 | vault:0 | ids:0 | ruview:0p # Daemon módban futtatás (háttérben, 30s-ként) nohup python3 ~/sb_push.py >> /tmp/sb-push.log 2>&1 & # VAGY: cron percenként (crontab -e) */1 * * * * python3 ~/sb_push.py --once >> /tmp/sb-push.log 2>&1
